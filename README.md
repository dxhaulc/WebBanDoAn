- Đăng nhập
- Xem danh sách đồ ăn theo tên, loại, giá

- Vào file "SQL Đồ án CNPM.sql" chạy để khởi tạo cơ sở dữ liệu.
- Vào terminal của thư mục code chạy node app.js
- Vào http://localhost:3000 đăng nhập tk: nv001 mk: password1
